# GOV.UK Figma Plugin - V2

## About
This plugin takes a GOV.UK page URL and automatically recreates it as a Figma frame using real design system components.

## How it works
1. HTML scraping — it fetches the real page markup via a Cloudflare Worker proxy, then walks the DOM to extract every GOV.UK (`govuk-*`) and publishing (`gem-c-*`) CSS class in order. Hidden and off-screen elements are skipped. Each component is also tagged with its grid column context (e.g. two-thirds, one-third) and any GOV.UK spacing utility classes applied to it.
2. Component matching — the plugin looks up each extracted class in a pre-built map (`component-map.json`) that links CSS classes to actual Figma component keys. This map was generated by `scanner.js`, which scanned both the GOV.UK Design System and GOV.UK Publishing Components Figma libraries.
3. AI ordering — a screenshot of the page is taken via thum.io and sent to Claude along with the matched component list. Claude uses the screenshot to sort and filter the list into the correct visual order, top to bottom.
4. Frame building — the matched components are imported from the Figma libraries and placed into a 1440px AutoLayout frame. Grid column context is used to split the body into the correct column layout (e.g. two-thirds main content, one-third sidebar). GOV.UK spacing variables from `spacing-map.json` are applied to gaps between components where they were present in the original markup.

### Improvement over V1
Unlike V1 (banana), which relied entirely on Claude to identify components from a screenshot, cherry reads the actual page markup. This means the component list is always accurate — Claude is only used to order it, not guess it. Components that are subtle, off-screen, or visually similar are no longer missed.

## How to run
1. Download this repo
2. Open Figma (ideally desktop version)
3. Select *Plugins* in the menu
4. Hover over *Development*
5. Choose the option *Import plugin from manifest*
6. Select the folder of downloaded repo
7. Select `manifest.json`
8. A pop-up should appear that contains all the plugins you've installed
9. Select *GOV.UK Page Builder (Cherry)*
10. A little software will appear where you will need a URL of a GOV.UK page and an [Anthropic API Key](https://platform.claude.com/).
11. Lastly, select the *Build page* button and then **hold on to your butts...**
